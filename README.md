This project relies on the SDL2 API. To install the necessary SDL2 dependencies run:<br>
```shell
sudo apt-get install build-essential libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev libsdl2-mixer-dev
```

Then to compile run `make` and right after `./main` and you will be playing the game.
